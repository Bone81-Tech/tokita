# File-file Penting dalam Proyek Tokita

## File Utama
- `index.html` - Halaman beranda utama
- `products.html` - Halaman katalog produk
- `styles.css` - File CSS utama dengan semua styling

## Folder dan Isinya
### `js/` - File JavaScript
- `config.js` - Konfigurasi Supabase
- `api.js` - Fungsi-fungsi untuk berinteraksi dengan Supabase
- `navigation.js` - Komponen navigasi dengan mobile menu
- `hero.js` - Komponen hero banner
- `footer.js` - Komponen footer
- `product-grid.js` - Komponen katalog produk dengan filter
- `product-card.js` - Komponen kartu produk
- `pwa-install.js` - Fungsi PWA install button
- `sw-cleanup.js` - Fungsi pembersihan service worker

### `images/` - Gambar-gambar
- Logo dan gambar untuk section about

### `icons/` - Icon PWA
- Icon untuk berbagai ukuran perangkat

## File PWA
- `manifest.json` - Konfigurasi PWA
- `sw.js` - Service worker
- `workbox-*.js` - File workbox untuk caching

## File Konfigurasi dan Dokumentasi
- `README.md` - Panduan utama proyek
- `docs/` - Dokumentasi tambahan
- `netlify.toml` - Konfigurasi deployment Netlify
- `.env.example` - Contoh konfigurasi environment
- `.gitignore` - Daftar file yang diabaikan oleh git
